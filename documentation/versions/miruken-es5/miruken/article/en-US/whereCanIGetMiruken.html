Where Can I Get Miruken-ES5?¶

The miruken-es5 package is just the miruken library.

The miruken-es5-angular package is the miruken library and the angular integration.

Yarn¶

We love Yarn so much and recommend you use it too!

yarn add miruken-es5

or

yarn add miruken-es5-angular

NPM¶

npm install miruken-es5

or

npm install miruken-es5-angular

Bower¶

Bower works, but even they recommend you use Yarn:)

bower install miruken-es5

or

bower install miruken-es5-angular
